## 相机
对于新手来说，相机大光圈是个很好用的东西，有了大光圈，场景不是最重要，这时候搭配最关键。
### 1.环境中的色彩的利用
1.环境中的色彩是营造画面的利器，在日常逛街的时候，可以多留意马路边**大面积色块**的场景。如果大场景的色块让环境显得杂乱，可以选择拍摄近景人像。
小广场上的一辆黄色的维修车。这个主色调黄色的车身加上点缀的其他颜色，为拍摄提供了很好的色彩对比。车尾箱的踏板可以让女朋友坐下凹造型，不会显得过于单调。

### 2.拍人像
在拍人像的时候，有时候**不必一味的正面构图**。侧面的拍摄加上树叶的前景，可以给相片添加很多神秘感，也不会让相片显得太过单一。以树叶做前景，这时候要注意的是：勿留太多前景，不然显得喧宾夺主。

### 3.夜景
夜景人像拍摄最重要的是**寻找光源**，这可以让人物轮廓清晰可见。晚上亮起的霓虹灯，大面积的背光源都是很好的场景。

我们找到的是一个古建筑的场景，它自带光源，而且拥有丰富的背景色。但是杂乱的人群让场景显得很乱，这时候要做的就是尝试局部构图，低角度的仰拍，既避开杂乱的人群，又可以让人物脸部显得更加立体。

### 4.拍照姿势

首先，你得知道你想拍的是以**大场景**为主的人像照还是**近景人像照**；如果是前者，其实拍照姿势真的一点都不重要，哪怕拍照的时候你呲牙裂嘴，因为人足够小，估计也不看出来。这时候要做的就是不用刻意摆拍，与场景的互动；

<img src="https://pic2.zhimg.com/80/v2-86a3d6f604567554596bfc135c649486_720w.jpg?source=1940ef5c" height="500px">

如果你一看镜头就会不知所措，那旧干脆不看呗。回眸，撩头发，推门，肆意的走或跑，借助前景，这些都是不错的拍照姿势，而且一点也不会显得做作。切记，如果脸不是非常非常好看，最好放弃大正脸拍过去，因为拍了那么久的妹纸，真的没几个敢驾驭没任何遮挡的正脸出镜。千万不要每次一到拍照的时候就让模特看镜头，这样拍出来反而很做作。

## 手机

### 基础知识
首先要清楚这些：

1、手机自带镜头是一个定焦的广角镜头。广角会形成画面畸变，这种变形本是镜头的“缺陷”。但如果掌握广角“畸变”的特性，比如低角度拍摄人像，分分钟拍出大长腿；当然，用不好只会拍出怪异的画面。

2、手机的曝光对焦快门趋于自动化，所以对焦，曝光尽量以人物脸部为准。但是对于一些大光比环境（画面明暗对比强烈）对焦在人物脸部时，脸部曝光准确，其他地方很容易过曝或过暗。这时候应该首选让画面高光不被溢出，人脸可以稍暗一点，采用后期拉高阴影，实现人脸曝光准确。

3.手机无法达到相机的景深效果（画面背景虚化程度），所以尽量避免近景拍人像。当然你也可以买个有人像模式的爪机（比如iPhone7p、8p x  或者大部分安卓手机的人像模式）或借助手机外置人像镜头，达到虚化效果。

### 1.手机构图

在保证手机曝光，快门和对焦熟练掌握的基础上，手机摄影另一个很重要的因素就是**构图**

因为手机的画质和景深限制，手机摄影应该把重心放在构图上。在拍摄过程中可以多留意人物的位置，让人物成为画面主体，通过不同的引导方式，让看相片的人能一下子注意人物。

### 2.光线

合理借助光线，不用露脸也可以拍的很好看

利用好天然的光线，或者找明暗对比强烈的场景，背对光源，拍出剪影效果。这时候，你可以让模特尽情的用各种姿势来凸显自己的曲线啦。无论是阳光，窗帘还是路边的灯光，都可以营造出一种温润，恬静的氛围。
