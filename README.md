### 1.环境中的色彩的利用
1.环境中的色彩是营造画面的利器，在日常逛街的时候，可以多留意马路边大面积色块的场景。如果大场景的色块让环境显得杂乱，可以选择拍摄近景人像。
小广场上的一辆黄色的维修车。这个主色调黄色的车身加上点缀的其他颜色，为拍摄提供了很好的色彩对比。车尾箱的踏板可以让女朋友坐下凹造型，不会显得过于单调。
